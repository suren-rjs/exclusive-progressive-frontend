# DSA Roadmap: Your Ultimate Guide for Data Structures and Algorithms

A comprehensive, interactive mindmap-style web application designed to help developers master Data Structures and Algorithms. This tool is specifically tailored to the patterns and problem sets required to crack technical interviews at top-tier companies like Amazon. 



## 🚀 Features

- **Mindmap-Style Layout**: Visually intuitive organization of DSA topics.
- **8 Comprehensive Mastery Sections**:
    - **I. Arrays & Hashing**: Foundational data manipulation.
    - **II. Two Pointers & Sliding Window**: Mastering "Caterpillar" and "Collision" logic.
    - **III. Stacks & Monotonic Patterns**: Solving "Next Greater/Smaller" challenges.
    - **IV. Binary Search**: Achieving $O(\log N)$ efficiency.
    - **V. Linked Lists**: Navigating node-based structures.
    - **VI. Trees & Graphs**: Hierarchical and network data traversal.
    - **VII. Dynamic Programming & Greedy**: High-level optimization strategies.
    - **VIII. Advanced Patterns**: Prefix Sums, Heaps, and Line Sweeps.
- **Real-time Progress Tracking**: Persistent stats and per-topic completion percentages saved via `localStorage`.
- **Curated Problem Set**: 100+ "must-do" LeetCode problems identified as core interview.
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack & Architecture

- **React + TypeScript**: Ensures type-safe, component-based development using modern.
- **SOLID Principles**: Focused on a clean, maintainable, and decoupled component structure.
- **Browser Persistence**: Uses `localStorage` for progress tracking without needing a backend.
- **Modern UI**: Gradient-based interface with smooth animations and responsive CSS.
